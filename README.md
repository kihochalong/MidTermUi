# MidTermUi

本儲存庫包含 Qt UI Design Studio 的完整使用案例文件，展示其在實際軟體開發場景中的應用。

## 內容

- **Qt_UI_Design_Studio_Use_Case.md** - 詳細的使用案例（661 字），描述使用 Qt Design Studio 開發智慧家居控制儀表板，涵蓋從設計原型到最終實作的完整工作流程。

## 使用案例概述

本文件呈現一個真實場景，SmartLiving Technologies 為 Android 平板電腦和嵌入式 Linux 設備開發 HomeHub 中央控制器。它展示了：

- 設計師與開發人員的協作工作流程
- Qt Design Studio 的功能與能力
- 4 週專案時間表分解
- 自訂 UI 元件的實作
- 跨平台開發方法
- 結果與經驗總結

此使用案例作為理解 Qt Design Studio 如何融入現代物聯網與嵌入式系統 UI/UX 開發流程的實用範例。
